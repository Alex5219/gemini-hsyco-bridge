Assistant Raspberry is a personal project developed by Alessandro Marciante to integrate smart home devices with Google Assistant. This Privacy Policy describes how user information is handled.

1. Data Collected

Assistant Raspberry does not collect or store any personally identifiable information. However, during the use of the service, Google may share technical information necessary for operation (e.g., user identifiers, device status).

2. Use of Data

  • Synchronize smart home devices
  • Execute voice commands via Google Assistant
  • Send state updates to Google (Report State)

3. Data Retention

We do not store any personal data on external servers. All information is handled in temporary memory and is not archived.

4. Data Sharing

We do not share any data with third parties. The data is used solely to provide the requested service through Google Assistant.

5. Security

We apply basic technical measures to protect communications between our server and Google, including HTTPS and OAuth 2.0 authentication.
